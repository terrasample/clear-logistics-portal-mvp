Backend API for Clear Logistics & Freight Services portal MVP.

Quick start:
1) Copy .env.example to .env and fill values as needed.
2) Run: npm install
3) Start API: npm run dev:backend

Endpoints:
- GET /api/health
- POST /api/accounts
- POST /api/login
- POST /api/quotes
- POST /api/bookings
- GET /api/shipments/:shipmentId
- POST /api/support
- POST /api/payments/checkout
